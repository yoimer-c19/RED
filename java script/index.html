Introducción al Desarrollo Web

El desarrollo web moderno se basa en tres tecnologías fundamentales que trabajan juntas para crear experiencias interactivas y atractivas en internet.

HTML - La estructura

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define la estructura y el contenido mediante un sistema de etiquetas.

  • Creado por Tim Berners-Lee en 1991
  • Versión actual: HTML5 (desde 2014)
  • Es un estándar mantenido por el W3C

CSS - El diseño

CSS (Cascading Style Sheets) controla la presentación visual de los documentos HTML, permitiendo separar el contenido de su apariencia.

  • Propuesto por Håkon Wium Lie en 1994
  • Versión actual: CSS3 (modular)
  • Permite diseños responsivos para diferentes dispositivos

JavaScript - La interactividad

JavaScript es un lenguaje de programación que permite crear contenido dinámico y responder a las acciones del usuario.

  • Creado por Brendan Eich en 1995
  • Estándar ECMAScript (versión actual: ES2022)
  • No debe confundirse con Java (son lenguajes diferentes)

¿Sabías que?

Estas tres tecnologías son interpretadas por el navegador web del cliente, lo que significa que no necesitan compilación previa y cualquier persona puede ver el código fuente de cualquier página web.

HTML: La estructura de la web

HTML es el lenguaje fundamental para crear páginas web. Consiste en una serie de elementos que definen la estructura y el significado del contenido.

Conceptos básicos

  • Elementos: Componentes básicos de HTML definidos por etiquetas (<etiqueta>contenido</etiqueta>)
  • Atributos: Configuraciones adicionales para elementos (ej: <a href="...">)
  • Estructura básica: Doctype, html, head (metadatos), body (contenido visible)
  • Semántica: Uso de elementos que describen su contenido (header, nav, section, article, footer)

Elementos HTML esenciales

Elemento Descripción Ejemplo
<h1>-<h6> Encabezados <h1>Título principal</h1>
<p> Párrafo <p>Texto de ejemplo.</p>
<a> Enlace <a href="https://ejemplo.com">Enlace</a>
<img> Imagen <img src="imagen.jpg" alt="Descripción">
<ul>, <ol>, <li> Listas <ul><li>Item 1</li></ul>
<div> Contenedor genérico <div>Contenido</div>

¡Importante!

Desde HTML5, se recomienda el uso de elementos semánticos como <header>, <nav>, <main>, <section>, <article>, <aside> y <footer> en lugar de <div> siempre que sea posible, ya que mejoran la accesibilidad y el SEO.

Estructura completa de un documento HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
    <meta name="description" content="Descripción de la página">
    <link rel="stylesheet" href="estilos.css">
</head>
<body>
    <header>
        <h1>Mi Sitio Web</h1>
        <nav>
            <ul>
                <li><a href="#inicio">Inicio</a></li>
                <li><a href="#servicios">Servicios</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section id="inicio">
            <h2>Bienvenido</h2>
            <p>Contenido de la sección...</p>
        </section>
    </main>
    
    <footer>
        <p>© 2023 Mi Sitio Web</p>
    </footer>
    
    <script src="script.js"></script>
</body>
</html>

CSS: Estilo y diseño web

CSS (Cascading Style Sheets) es el lenguaje que controla la presentación visual de los documentos HTML, permitiendo crear diseños atractivos y responsivos.

Fundamentos de CSS

  • Selectores: Patrones que identifican a qué elementos se aplicarán los estilos
  • Propiedades: Características visuales que se pueden modificar (color, tamaño, posición, etc.)
  • Valores: Configuraciones específicas para cada propiedad
  • Cascada: Orden en que se aplican los estilos cuando hay conflictos
  • Especificidad: Reglas que determinan qué estilos tienen prioridad

Formas de aplicar CSS

Estilos aplicados directamente en el elemento mediante el atributo style:

<p style="color: blue; font-size: 16px;">
    Este texto será azul y de 16px.
</p>

No recomendado: Difícil de mantener y mezcla contenido con presentación.

Estilos definidos en la sección <head> del documento HTML:

<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>

Útil para: Estilos específicos de una sola página.

Estilos en archivo separado vinculado con <link>:

<head>
    <link rel="stylesheet" href="estilos.css">
</head>

Contenido de estilos.css:

p {
    color: blue;
    font-size: 16px;
}

Mejor práctica: Permite reutilizar estilos en múltiples páginas y mejora el rendimiento.

Selectores CSS comunes

Selector Ejemplo Descripción
Selector de elemento p { } Selecciona todos los elementos <p>
Selector de clase .clase { } Selecciona elementos con class="clase"
Selector de ID #id { } Selecciona el elemento con id="id"
Selector de atributo [type="text"] { } Selecciona elementos con el atributo especificado
Selector descendente div p { } Selecciona todos los <p> dentro de <div>
Selector de pseudo-clase a:hover { } Selecciona elementos en estado específico

Modelo de caja (Box Model)

Concepto fundamental en CSS que describe cómo se representan visualmente los elementos:

Diagrama del modelo de caja CSS (Fuente: MDN)

  • Contenido (content): El área donde se muestra el contenido (texto, imágenes, etc.)
  • Relleno (padding): Espacio entre el contenido y el borde
  • Borde (border): Línea que rodea el padding y el contenido
  • Margen (margin): Espacio exterior que separa el elemento de otros elementos

Flexbox y Grid: Diseños modernos

CSS ofrece dos poderosos sistemas para crear diseños complejos y responsivos:

Flexbox es un modelo unidimensional para distribuir elementos en una fila o columna:

.contenedor {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1;
    margin: 10px;
}

JavaScript: Programación para la web

JavaScript es un lenguaje de programación que permite crear contenido dinámico, interactuar con el usuario y manipular el DOM.

Conceptos fundamentales

  • Variables: Almacenan datos (let, const, var)
  • Tipos de datos: Números, strings, booleanos, objetos, arrays, etc.
  • Operadores: Aritméticos, comparación, lógicos
  • Estructuras de control: Condicionales (if, switch), bucles (for, while)
  • Funciones: Bloques de código reutilizables
  • Objetos: Colecciones de propiedades y métodos
  • Eventos: Respuesta a acciones del usuario

Sintaxis básica

// Variables
let nombre = "Juan";
const edad = 30;

// Función
function saludar(persona) {
    return "Hola, " + persona;
}

// Condicional
if (edad >= 18) {
    console.log("Mayor de edad");
} else {
    console.log("Menor de edad");
}

// Evento
document.getElementById("boton").addEventListener("click", function() {
    alert("Botón clickeado!");
});

Manipulación del DOM

El DOM (Document Object Model) es una representación del documento HTML que JavaScript puede manipular:

Método Ejemplo Descripción
Seleccionar elemento document.querySelector('#id') Selecciona el primer elemento que coincida
Seleccionar múltiples document.querySelectorAll('.clase') Selecciona todos los elementos que coincidan
Cambiar contenido elemento.textContent = 'texto' Modifica el texto de un elemento
Cambiar HTML elemento.innerHTML = '<strong>texto</strong>' Modifica el HTML interno
Cambiar estilos elemento.style.color = 'red' Modifica estilos CSS
Agregar clase elemento.classList.add('clase') Añade una clase CSS

JavaScript moderno (ES6+)

Las versiones recientes de JavaScript introdujeron importantes mejoras:

  • Let/Const: let y const en lugar de var
  • Arrow functions: () => {} Sintaxis más corta para funciones
  • Template literals: `Hola ${nombre}` para strings con variables
  • Destructuring: const {prop} = objeto para extraer propiedades
  • Spread operator: [...array] para copiar arrays/objetos
  • Classes: Sintaxis más clara para programación orientada a objetos

Manejo moderno de operaciones asíncronas:

// Promesas
fetch('https://api.ejemplo.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// Async/Await (más legible)
async function obtenerDatos() {
    try {
        const response = await fetch('https://api.ejemplo.com/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}

Organización de código en módulos reutilizables:

// math.js
export function sumar(a, b) {
    return a + b;
}

// app.js
import { sumar } from './math.js';
console.log(sumar(2, 3)); // 5

Beneficios: Código más organizado, reutilizable y mantenible.

Ejercicio 3: Añadir interactividad con JavaScript

Añade funcionalidad JavaScript a la página:

  1. Crea un botón que muestre/oculte la sección de películas
  2. Implementa un contador de visitas que se muestre en el pie de página
  3. Añade un formulario de contacto con validación básica
  4. Crea un reloj en tiempo real que se actualice cada segundo
  5. Implementa un cambio de tema (claro/oscuro)
// 1. Botón para mostrar/ocultar películas
document.getElementById('toggle-movies').addEventListener('click', function() {
    const moviesSection = document.getElementById('movies-section');
    if (moviesSection.style.display === 'none') {
        moviesSection.style.display = 'block';
        this.textContent = 'Ocultar películas';
    } else {
        moviesSection.style.display = 'none';
        this.textContent = 'Mostrar películas';
    }
});

// 2. Contador de visitas (usando localStorage)
if (!localStorage.visitCount) {
    localStorage.visitCount = 0;
}
localStorage.visitCount++;
document.getElementById('visit-count').textContent = localStorage.visitCount;

// 3. Validación de formulario
document.getElementById('contact-form').addEventListener('submit', function(e) {
    const email = document.getElementById('email').value;
    if (!email.includes('@')) {
        e.preventDefault();
        alert('Por favor ingresa un email válido');
    }
});

// 4. Reloj en tiempo real
function updateClock() {
    const now = new Date();
    document.getElementById('clock').textContent = now.toLocaleTimeString();
}
setInterval(updateClock, 1000);
updateClock();

// 5. Cambio de tema
document.getElementById('theme-toggle').addEventListener('click', function() {
    document.body.classList.toggle('dark-theme');
    this.textContent = document.body.classList.contains('dark-theme') ? 
        'Tema claro' : 'Tema oscuro';
});

/* CSS adicional para el tema oscuro */
.dark-theme {
    background-color: #222;
    color: #eee;
}

.dark-theme main {
    background-color: #333;
    color: #eee;
}

Ejercicio 4: Proyecto integrador

Crea una página web completa que incluya:

  1. Estructura semántica HTML5
  2. Diseño responsivo con Flexbox o Grid
  3. Estilos CSS organizados
  4. Interactividad con JavaScript
  5. Un formulario funcional
  6. Efectos de hover/transiciones
  7. Integración con una API pública (como clima o noticias)

Sugerencia: Podría ser un portafolio personal, un blog o una página de producto.

Este es un proyecto más complejo que integra todos los conceptos. Aquí hay una estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <link rel="stylesheet" href="styles.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">

<body>
    <header>
        <nav>
            <div class="logo">MiPortafolio</div>
            <ul class="nav-links">
                <li><a href="#inicio">Inicio</a></li>
                <li><a href="#proyectos">Proyectos</a></li>
                <li><a href="#contacto">Contacto</a></li>
            </ul>

        </nav>
    </header>
    
    <main>
        <section id="inicio" class="hero">
            <div class="hero-content">
                <h1>Hola, soy <span class="highlight">Desarrollador Web</span></h1>
                <p>Creo experiencias digitales impactantes</p>
                <a href="#proyectos" class="btn">Ver mis trabajos</a>
            </div>
            <div class="hero-image">
                <img src="developer.svg" alt="Desarrollador web">
            </div>
        </section>
        
        <section id="proyectos">
            <h2>Mis Proyectos</h2>
            <div class="projects-grid">
                <!-- Proyectos cargados dinámicamente con JavaScript -->
            </div>
        </section>
        
        <section id="contacto">
            <h2>Contacto</h2>
            <form id="contact-form">
                <input type="text" placeholder="Nombre" required>
                <input type="email" placeholder="Email" required>
                <textarea placeholder="Mensaje" required></textarea>
                <button type="submit" class="btn">Enviar</button>
            </form>
        </section>
    </main>
    
    <footer>
        <p>© 2023 Mi Portafolio. Visitas: <span id="visit-count">0</span></p>
        <div class="social-links">
            <a href="#"><i class="fab fa-github"></i></a>
            <a href="#"><i class="fab fa-linkedin"></i></a>
        </div>

    <script src="script.js"></script>
</body>
</html>
